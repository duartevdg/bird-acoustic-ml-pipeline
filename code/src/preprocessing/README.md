# Preprocessing Folder

This folder contains modules for audio preprocessing used in the project.

Included modules:

- **amplitude.py** – Applies amplitude scaling and normalization methods.  
- **augmentation.py** – Performs data augmentation techniques for audio recordings.  
- **denoising.py** – Implements noise-reduction methods.  
- **normalisation.py** – Handles signal normalization and standardization procedures.  
- **preemphasis.py** – Applies pre-emphasis filtering to enhance high-frequency components.  
- **segmentation.py** – Segments recordings into time-based or event-based clips.  
- **__init__.py** – Package initializer for preprocessing module imports.
