# Dataset Folder

This folder contains scripts for creating and managing datasets.

Included scripts:

- **dataset_creation.py** – Constructs and organizes the dataset from raw recordings.  
- **fetch_metadata.py** – Retrieves and processes metadata associated with recordings.  
- **group_info.py** – Organizes recordings and metadata into groups for analysis.  
- **subset_creation.py** – Creates dataset subsets for experiments, training, and evaluation.
