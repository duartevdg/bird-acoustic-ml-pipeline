# Feature Extraction Folder

This folder contains modules for extracting and normalizing audio features for the project.

Included modules:

- **batch_extract.py** – Orchestrates batch feature extraction across datasets.  
- **time_domain.py** – Extracts time-domain signal features.  
- **frequency_domain.py** – Computes frequency-domain and spectral features.  
- **time_frequency.py** – Generates time–frequency representations and related features.  
- **cepstral_domain.py** – Extracts cepstral features.  
- **feature_normalization.py** – Applies normalization and scaling to extracted feature sets.  
- **__init__.py** – Package initializer for module imports.
