# Utils Folder

This folder contains utility modules that support common operations across the project.

Included modules:

- **file_utils.py** – Handles file management, directory operations, and input/output utilities.  
- **helpers.py** – Provides general helper functions used throughout the pipeline.  
- **metadata.py** – Processes, validates, and manages recording metadata.  
- **__init__.py** – Package initializer for utility module imports.
