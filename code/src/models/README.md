# Models Folder

This folder contains modules for building, training, tuning, and evaluating machine-learning models.

Included modules:

- **base.py** – Defines shared model interfaces and common utilities.  
- **classifiers.py** – Implements machine-learning classifiers used in the pipeline.  
- **train.py** – Handles model training workflows.  
- **tune.py** – Performs hyperparameter optimization.  
- **evaluate.py** – Computes evaluation metrics and generates performance reports.  
- **__init__.py** – Package initializer for model module imports.
